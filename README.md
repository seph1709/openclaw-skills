# OpenClaw Skills & Automations

A compilation of all OpenClaw-related skills, automations, and tools built for the [OpenClaw](https://github.com/seph1709/office-openclaw) platform — covering social media automation, developer productivity, AI integrations, and more.

---

## Skills

| Repo | Description |
|------|-------------|
| [skill-extractor](https://github.com/seph1709/skill-extractor) | Export any installed skill into a shareable ZIP with STRUCTURE.md |
| [clawhub-skill-creator](https://github.com/seph1709/clawhub-skill-creator) | Scaffold ClawHub-ready skills with correct structure, scanner criteria & security rules |
| [facebook-page](https://github.com/seph1709/facebook-page) | Interact with any Facebook Page via Meta Graph API |
| [fb-page](https://github.com/seph1709/fb-page) | Automate Facebook Page posts, scheduling, messaging, and more via the Meta Graph API |
| [fb-inbox-forward](https://github.com/seph1709/fb-inbox-forward) | Poll Facebook Page inbox and forward new messages to a configured notification channel |
| [instagram-page](https://github.com/seph1709/instagram-page) | Interact with any Instagram Business/Creator account via Instagram Graph API |
| [tiktok-page](https://github.com/seph1709/tiktok-page) | Interact with any TikTok account feature via the TikTok Content Posting API |
| [x-twitter](https://github.com/seph1709/x-twitter) | Interact with any X/Twitter account via X API v2 |
