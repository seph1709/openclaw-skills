# My OpenClaw Skills & Tools

Personal collection of custom skills and automation tools I built for the OpenClaw platform — from social media integrations to developer utilities.

---

## Custom Skills

| Skill | Description |
|-------|-------------|
| [fb-page](https://github.com/seph1709/fb-page) | Automate Facebook Page posts, scheduling, and messaging via Meta Graph API |
| [fb-inbox-forward](https://github.com/seph1709/fb-inbox-forward) | Poll Facebook Page inbox and forward new messages to a notification channel |
| [instagram-page](https://github.com/seph1709/instagram-page) | Interact with any Instagram Business/Creator account via Instagram Graph API |
| [tiktok-page](https://github.com/seph1709/tiktok-page) | Interact with any TikTok account via the TikTok Content Posting API |
| [x-twitter](https://github.com/seph1709/x-twitter) | Interact with any X/Twitter account via X API v2 |
| [skill-extractor](https://github.com/seph1709/skill-extractor) | Export any installed skill into a shareable ZIP with STRUCTURE.md |
| [clawhub-skill-creator](https://github.com/seph1709/clawhub-skill-creator) | Scaffold ClawHub-ready skills with correct structure, scanner criteria & security rules |
